01背包

问题描述

有$N$件物品和一个容量是$V$的背包。每件物品只能使用一次。

第$i$件物品的体积是 $v_i$，价值是 $w_i$。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。
输出最大价值。

输入格式
第一行两个整数，$N$，$V$，用空格隔开，分别表示物品数量和背包容积。
接下来有$N$行，每行两个整数 $v_i$, $w_i$，用空格隔开，分别表示第$i$件物品的体积和价值

输出格式
输出一个整数，表示最大价值。

选还是不选

DP分析

  1. 状态表示$f[i][j]$
    选法问题：第一个维度表示选哪个物品，后一个维度表示是哪个限制。
    • 集合：所有只考虑前i个物品，且总体积不超过j的选法的集合。
    • 属性：$f[i][j]$表示最大价值
  2. 状态计算
    选 或者 不选这个物品。
    1. 所有不选第i个物品的方案。 $f[i][j] = f[i - 1][j]$
    2. 所有选第i个物品的方案。$f[i][j] = f[i][j - v[i]] + w[i]$
      • 变化的部分：前$i-1$个物品随便选。变化的部分价值最大是：从$1$~${i-1}$中选，总体积$<=j-v[i]$;即$f[i - 1][j - v[i]]$
      • 不变的部分：第$i$个物品必须选。价值固定是$w[i]$

状态计算的原则：

  1. 不重复
  2. 不遗漏

二维动态规划

$f[i][j]$ 表示只看前i个物品，总体积是j的情况下，总价值最大是多少？
$result = max{f[n][0-V]}$

$f[i][j]$的计算

  1. 不选第i个物品，$f[i][j] = f[i - 1][j];$
  2. 选第i个物品，$f[i][j] = f[i-1][j - v[i]] + w[i];$

$f[i][j] = max(1, 2)$

$f[i][j]$初始化

$f[0][0] = 0$; 没有选任何物品，体积是0的情况下，最大价值是0

时间复杂度 $O(n^2)$

一维动态规划

空间上进行优化
$f[i][j] = max(f[i - 1][j], f[i - 1][j - v[i]] + w[i]);$

  • 所有$f[i]$只会用到$f[i-1]$
    可以用一个滚动数组来求，每次只保留上层的结果
  • j这层每次用j或者比j小的数$j-v[i]$.可以优化成一个数组。循环时从大到小循环：
    • $f[j] = max(f[j], f[j-v[i]] + w[i])$由于从大到小循环，此时$f[j-v[i]]$还有没被第i层更新，是上层的$f[i-1]$层的数据

完全背包

问题描述

有$N$种物品和一个容量是$V$的背包，每种物品都有无限件可用。
第$i$种物品的体积是 $v_i$，价值是 $w_i$。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。输出最大价值。

输入格式
第一行两个整数，$N$，$V$，用空格隔开，分别表示物品种数和背包容积。
接下来有$N$行，每行两个整数$v_i, w_i$，用空格隔开，分别表示第 $i$种物品的体积和价值。

输出格式
输出一个整数，表示最大价值。

DP分析

  1. 状态表示$f[i][j]$
    选法问题：第一个维度表示选哪个物品，后一个维度表示是哪个限制。
    • 集合：所有只考虑前i个物品，且总体积不超过j的选法的集合。
    • 属性：$f[i][j]$表示最大价值
  2. 状态计算
    $f[i][j] = max(f[i - 1][j], f[i-1][j-v]+w, …, f[i-1][j-kv]+kw, …)$;共有$n$项
    • 选0个i物品的集合 $f[i - 1][j]$
    • 选1个i物品的集合
    • …
    • 选$k$个$i$物品的集合 $f[i - 1][j - k * v[i]] + k * w[i]$
    • …

状态计算的数学推导：

$$
f[i][j] = max(f[i - 1][j], f[i-1][j-v]+w, …, f[i-1][j-kv]+kw, …);\
f[i][j-v] = max(f[i-1][j-v], f[i-1][j-2v]+w, f[i-1][j-3v]+2w, …);$$
所以：
$f[i][j] = max(f[i-1][j], f[i][j-v] + w);$

01背包与完全背包数学公式的对比

  1. 01背包：$f[i][j] = max(f[i-1][j], f[i-1][j-v]+w)$
  2. 完全背包：$f[i][j] = max(f[i - 1][j], f[i][j - v] + w);$
    正是这个区别，如果是优化空间的话，一个是从大到小枚举体积，另一个是从小到大枚举体积。
